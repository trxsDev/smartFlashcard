# รายการกราฟิก UI (Image Assets) ที่ต้องใช้

เพื่อเปลี่ยนจาก UI เรขาคณิตแข็งๆ ให้กลายเป็นเกมที่สวยงามเหมือน Candy Crush คุณสามารถออกแบบรูปภาพเหล่านี้ด้วยโปรแกรมภายนอก (Figma, Photoshop, Illustrator) แล้วเซฟเป็นชิ้นๆ มาวางทับโครงสร้างเดิมได้เลยครับ

กรุณาเซฟรูปเป็นสกุลไฟล **.png (พื้นหลังใส)** แล้วนำมาใส่ตามโฟลเดอร์ด้านล่างนี้:

## 1. กลุ่มภาพพื้นหลัง (`assets/ui/backgrounds/`)

ภาพทั้งหมดในกลุ่มนี้ควรมีขนาด **1280 x 720 px** (แนวนอน)

- `bg_menu.png` - ภาพพื้นหลังสำหรับหน้า Settings และหน้า Landing Page (เลือกบทเรียน) ควรมีลายที่สดใสและน่ารัก
- `bg_gameplay.png` - ภาพพื้นหลังตอนกำลังเล่นเกม (มีกล้อง) ควรเว้นที่ว่างด้านบนไว้สำหรับแถบข้อความยาวๆ

## 2. กลุ่มปุ่มกด (`assets/ui/buttons/`)

- `btn_primary.png` - ปุ่มกดยาวหลัก (เช่น "เลือกบทเรียน", "กลับหน้าหลัก") ขนาดประมาณ **300 x 80 px** สีโทนน่ารัก (เช่น ฟ้า/ชมพู)
- `btn_play.png` - ปุ่มเริ่มเล่นเกม ขนาดประมาณ **300 x 80 px** สีที่โดดเด่นตัดกับฉาก (เช่น ส้ม/เหลือง)
- `btn_arrow_left.png` - ปุ่มลูกศรซ้าย (สำหรับเปลี่ยนกล้อง) ขนาดประมาณ **60 x 60 px**
- `btn_arrow_right.png` - ปุ่มลูกศรขวา (สำหรับเปลี่ยนกล้อง) ขนาดประมาณ **60 x 60 px**

## 3. กลุ่มไอคอนสถานะ (`assets/ui/icons/`)

ควรเป็นภาพไอคอนโดดๆ พื้นหลังใส

- `heart_full.png` - ไอคอนหัวใจสีแดงเต็มดวง (แสดงพลังชีวิต) ขนาดประมาณ **50 x 50 px**
- `heart_empty.png` - ไอคอนหัวใจสีเทาหรือว่างเปล่า (แสดงการตอบผิด) ขนาดประมาณ **50 x 50 px**
- `lock_icon.png` - แม่กุญแจสำหรับด่านที่ยังไม่ปลดล็อค ขนาดประมาณ **80 x 80 px**
- `wifi_check.png` - เครื่องหมายติ๊กถูกการเชื่อมต่ออินเทอร์เน็ตผ่าน ขนาดประมาณ **100 x 100 px**

## 4. กลุ่มแผ่นป้าย (`assets/ui/panels/`)

- `panel_chapter_unlocked.png` - กรอบสี่เหลี่ยมโค้งๆ สำหรับด่านที่เล่นได้ (บทที่ 1) ขนาด **240 x 160 px**
- `panel_chapter_locked.png` - กรอบสี่เหลี่ยมโค้งๆ สีเทา สำหรับด่านจางๆ ขนาด **240 x 160 px**
